# Movie Verse

A simple React learning project built with Vite.

## App Preview

![Movie Verse Home](./screenshots/home.png)

Users can:
- Browse popular movies
- Search movies by title
- Save and remove favourites
- Click a movie card to open that movie on IMDb

## Tech Stack

- React
- React Router
- Vite
- TMDB API
- LocalStorage (for favourites)


## API

This app uses TMDB endpoints for:
- Popular movies
- Movie search
- External IDs (to resolve IMDb URL)


## Learning Focus

This project is intentionally simple and focused on core React concepts:
- Components and props
- State and effects
- Context API
- Routing
- Basic API integration
